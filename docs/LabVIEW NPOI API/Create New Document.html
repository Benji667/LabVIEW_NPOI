LabVIEW NPOI.lvlib:Document.lvclass:Create New Document.vi

Creates a new Document at the File Path location. Use save to persist your changes.

Notes:

- If File Path is empty (default) or is <Not A Path> or is a directory, the VI displays a dialog box from which the user can select a file. Error 43 occurs if you cancel the dialog box.

- If the File Path already contains a docuement with the same name then that file is overwritten without warning.
