Componant API

Open or Create and Close

The following VIs allow you to Open or Create a (new) document. The call of the Close function is mandatory to close any references that you open or create.

Open Document
Create New Document
Close Document

Append Elements

The following VIs allow you to append elements to the document.

Append Text
Append Table
Append Table (string)
Append Table (double)
Append Image

Retrieve Elements

The following VIs allow you to retrieve elements from the document.

Read Text
Read Table
Read Table As String
Read Table (string)
Read Table (double)

Document Properties

The following VIs allow you to access to the properties of the document.

Get FilePath
Set FilePath
Get Style Collection
Set Style Collection
Get Style List
Set Document Style

Excel Specific

The following VIs allow you to incorporate Microsoft Excel features into the document.

Add New Page
Remove Page

Save And Print

The following VIs allow you to save or print the document.

Save Document
Save As Document
Convert To HTML
Print Document
