The LabVIEW NPOI API exposes properties and methods to an abstract LVClass called Document. This allows us to have a unique manner to manipulate MS Office document, hiding the complexity of the underneath specific components for each file format (*.doc, *.docx, *.xls,*.xlsx). This Document class is a the top of the class hierarchy, that helps us to ensure reliability , scalability, maintainability of the whole library.

The LabVIEW NPOI dynamically handle the file format, based on the file extension that if pass through, to load the propper type of class. This will help the user to carry out tasks as efficiently as possible and to keep focuss on its most important topics.
