LabVIEW NPOI.lvlib:Document.lvclass:Create New Document.vi

Creates a new <b>Document</b> at the <b>File Path</b> location. Use save to persist your changes.

Notes:

- If <b>File Path</b> is empty (default) or is <Not A Path> or is a directory, the VI displays a dialog box from which the user can select a file. Error 43 occurs if you cancel the dialog box.

- If the <b>File Path</b> already contains a docuement with the same name then that file is overwritten without warning.

 error in (no error)
